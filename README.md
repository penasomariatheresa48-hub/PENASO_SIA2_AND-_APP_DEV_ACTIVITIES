# SIA 2 Activities

## Activity 1

Activity 1 focuses on creating a Food and Beverage Menu that showcases different meals and drinks with their corresponding categories, prices, and short descriptions to provide customers with clear and organized menu information.

## Activity 2

Activity 2 focuses on developing a Food Menu System that allows users to browse different menu items digitally, making menu viewing faster, easier, and more convenient for customers and administrators.
## Activity 3

Activity 3 focuses on creating a Student Form System that allows users to input, manage, and organize student information efficiently and accurately.
## Activity 4

Activity 4 focuses on creating UML Diagrams for the FOOD MENU RESTAURANT System to visually represent the interaction between users and the system, including customer and admin functions such as managing menu items, placing orders, and processing payments.
## Activity 5

Activity 5 focuses on developing a News Dashboard using Laravel that integrates authentication, APIs, and external data sources to display news information dynamically for users through an organized and interactive dashboard.
